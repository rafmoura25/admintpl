# 介绍
* 一整套基于bootstrap的网站后台静态模板，适合做后台管理系统的前端
* 最初基于国外的一套模板修改的，因为国外是英文版的，布局上很多都不太适合国内的风格
* 常用的表单、表格、图表、地图都有，常用的登录、注册、文章管理、数据大盘页面也都写得有
